# 🐾 Backend - Happy Paws Care

This is the backend server for Happy Paws Care application built with Node.js, Express, and MongoDB.

---

## 🛠️ Technologies Used

- 🟢 Node.js
- 🚂 Express.js
- 🍃 MongoDB (with Mongoose)
- 🔐 JWT Authentication
- 💳 Stripe Payment Gateway (Test Mode)

  
---
